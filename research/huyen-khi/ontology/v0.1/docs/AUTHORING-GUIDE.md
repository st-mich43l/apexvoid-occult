# Rule Authoring Guide

1. Identify a physical fact or exact combination.
2. Select an explicit school profile.
3. Attach a reviewed claim and locator.
4. State conditions.
5. Add symbolic effects by dimension.
6. Add stacking/conflict behavior.
7. Record limitations and contradictions.
8. Add or update an expert fixture.
9. Keep the rule disabled until review gates pass.

Forbidden: numeric weights, prior module axes, external output as doctrine,
fuzzy Tứ Hóa targets, school fallback, personal-chart special cases.
